## [1.1.0] - 2025-03-31

### 新增

- 🔒 **私有仓库支持**（功能 #40）
  - Personal Access Token AES-256-GCM 加密存储
  - 支持 GitHub / GitLab / Gitee
  - Token 一次配置，多个私有项目复用
  - Token 验证接口，部署前确认有效性
  - 侧边栏新增「🔒 私有仓库」管理页

- 🌍 **多环境远程主机部署**（功能 #26）
  - SSH 连接支持密码 + 私钥双认证
  - 自动识别远程项目类型，安装依赖后启动
  - 支持树莓派、云服务器、VPS、NAS 等任意 Linux 机器
  - 测试连接功能，确认 SSH 可用后再部署
  - 侧边栏新增「🌍 远程部署」页，主机 CRUD 管理

- ⚡ **事件驱动 Webhook 自动部署**（功能 #35）
  - HMAC-SHA256 签名验证（GitHub / GitLab 双格式）
  - 按分支过滤（只有指定分支 push 才触发）
  - 按事件过滤（push / release / pull_request）
  - 自动流水线：git pull → 安装依赖 → 重启服务
  - 侧边栏新增「⚡ 自动触发」页，Webhook 配置管理

- 🔗 **一键分享部署记录**（功能 #19）
  - AES 加密 token，防止伪造
  - 可设置有效期（24h / 3天 / 7天 / 30天 / 永久）
  - 包含项目配置 + 复现步骤
  - 分享页无需安装 GADA 即可查看
  - 侧边栏新增「🔗 分享记录」页

- 🃏 **项目卡片快捷按钮**
  - 💾 备份/回滚：弹窗查看历史备份，一键回滚到任意版本
  - 📋 部署日志：弹窗查看历次部署记录和输出
  - 🔗 分享：一键生成分享链接，弹窗内直接复制
  - 🌍 远程部署：弹窗选择目标主机，一键推送
  - 所有操作均在浏览器面板内完成，无需命令行

- 🎨 **UI 补全**
  - 新增 `.card` 通用卡片组件
  - 修复 `.modal-box` 使用了不存在的 `--bg1` 变量
  - 项目卡片按钮区自动换行，小屏下字号自适应
  - 侧边栏多分组时支持滚动，滚动条美化
  - 弹窗进入动画（slide + fade）
  - 分享页改为两栏 grid 布局
  - 响应式：768px 以下新功能卡片满宽

### 修复

- 项目卡片「🔄 重试」按钮 emoji 乱码
- 部署日志弹窗 `JSON.parse` 无容错导致报错（新增 `safeParseLogOutput()`）
- 日志输出未做 HTML 转义（XSS 防护）
- `.env.example` 缺少 `GADA_SECRET_KEY` 和 SSH 超时配置说明

### 依赖

- 新增 `node-ssh` ^13.x（远程 SSH 部署）

## [1.0.0] - 2024-03-30

### 新增
- 🎨 **UI v2.0** 全面重设计
  - 全新设计系统（阴影/渐变/动画）
  - 侧边栏项目计数 badge、AI 状态指示点
  - 首页快捷示例 chip、已安装项目快速入口
  - 功能亮点四宫格
  - 分析步骤三段动画
  - 部署进度百分比显示
  - 完成页弹出动画 + 访问地址一键复制
  - Toast 通知系统替代 alert
  - 项目卡片状态筛选
  - 响应式布局（手机/平板适配）
- 🔄 **git pull 更新**：项目卡片「⬇️ 更新」按钮
- 🔄 **失败重试**：失败项目「🔄 重试」按钮
- 🌐 **直接访问**：启动后显示「🌐 打开」链接
- 🏥 **健康检查**：每 30 秒轮询，连续 3 次失败发出告警
- ✏️ **备注/标签**：项目卡片支持添加备注和标签
- ⚡ **一键粘贴 AI 配置**：自动识别中转站各种格式
- 🗑️ **清除对话历史**：AI 对话弹窗新增清除按钮
- 📋 **日志复制**：部署日志一键复制
- ⚙️ **Settings API**：`GET/POST /api/settings/env` 读写 .env
- 🔒 **敏感配置保护**：`/api/config/:key` 屏蔽 API Key 读取

### 修复
- 进程管理 key 类型不一致导致进程状态查找失败
- AI 对话历史无限增长耗尽 token
- clone 无超时导致大仓库卡死
- 启动项目后端口未持久化到数据库
- .env settings 误匹配注释行
- scan.js 死代码 getQuickSize
- Object.assign(ConversationDB) 顺序问题
- tar 备份/回滚改用 spawn 数组参数防注入
- Docker compose 支持、Windows venv 路径修复

### 优化
- AI 提供商列表：已配置在前，未配置折叠
- getDirSize 改用 du 命令快速估算
- 克隆失败自动清理残留目录
- 对话历史限制最近 20 条
- 启动恢复进程状态（防幽灵 running 状态）
- WebSocket 日志缓存（200条），重连自动回放
