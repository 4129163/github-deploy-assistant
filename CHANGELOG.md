## [1.0.0] - 2024-03-30

### 新增
- 🎨 **UI v2.0** 全面重设计
  - 全新设计系统（阴影/渐变/动画）
  - 侧边栏项目计数 badge、AI 状态指示点
  - 首页快捷示例 chip、已安装项目快速入口
  - 功能亮点四宫格
  - 分析步骤三段动画
  - 部署进度百分比显示
  - 完成页弹出动画 + 访问地址一键复制
  - Toast 通知系统替代 alert
  - 项目卡片状态筛选
  - 响应式布局（手机/平板适配）
- 🔄 **git pull 更新**：项目卡片「⬇️ 更新」按钮
- 🔄 **失败重试**：失败项目「🔄 重试」按钮  
- 🌐 **直接访问**：启动后显示「🌐 打开」链接
- 🏥 **健康检查**：每 30 秒轮询，连续 3 次失败发出告警
- ✏️ **备注/标签**：项目卡片支持添加备注和标签
- ⚡ **一键粘贴 AI 配置**：自动识别中转站各种格式
- 🗑️ **清除对话历史**：AI 对话弹窗新增清除按钮
- 📋 **日志复制**：部署日志一键复制
- ⚙️ **Settings API**：`GET/POST /api/settings/env` 读写 .env
- 🔒 **敏感配置保护**：`/api/config/:key` 屏蔽 API Key 读取

### 修复
- 进程管理 key 类型不一致导致进程状态查找失败
- AI 对话历史无限增长耗尽 token
- clone 无超时导致大仓库卡死
- 启动项目后端口未持久化到数据库
- .env settings 误匹配注释行
- scan.js 死代码 getQuickSize
- Object.assign(ConversationDB) 顺序问题
- tar 备份/回滚改用 spawn 数组参数防注入
- Docker compose 支持、Windows venv 路径修复

### 优化
- AI 提供商列表：已配置在前，未配置折叠
- getDirSize 改用 du 命令快速估算
- 克隆失败自动清理残留目录
- 对话历史限制最近 20 条
- 启动恢复进程状态（防幽灵 running 状态）
- WebSocket 日志缓存（200条），重连自动回放
