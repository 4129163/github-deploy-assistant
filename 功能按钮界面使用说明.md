# GitHub Deploy Assistant 功能按钮界面使用说明

## 概述

为方便用户使用GitHub Deploy Assistant的所有功能，我们创建了一个全新的功能按钮界面。这个界面将所有复杂的命令行功能转化为直观的图形化按钮，让普通用户也能轻松使用。

## 访问方式

## 使用方法

### 基础使用
1. **选择分类**：在左侧边栏选择功能分类
2. **查看功能**：浏览该分类下的所有功能按钮
3. **点击按钮**：点击感兴趣的功能按钮
4. **查看命令**：右侧面板显示对应命令
5. **复制使用**：点击"复制"按钮使用命令

### 高级功能
1. **搜索功能**：在搜索框输入关键词快速查找
2. **主题切换**：点击月亮/太阳图标切换深色/浅色主题
3. **历史记录**：在右侧面板查看最近使用的命令
4. **帮助文档**：点击问号图标查看详细帮助

### 专业用户使用
1. **命令参考**：每个功能都显示完整的命令行命令
2. **参数提示**：命令中包含占位符，提示需要替换的参数
3. **批量操作**：可以同时查看多个功能的命令
4. **历史学习**：通过历史记录了解常用命令

### 扩展功能
要添加新功能，只需在JavaScript的`functionsData`对象中添加：
```javascript
{
    id: 'unique-id',
    title: '功能名称',
    description: '功能描述',
    command: 'gada command --options',
    icon: 'fas fa-icon-name',
    category: 'category-name'
}
```

### 自定义样式
可以通过修改CSS变量来定制界面：
```css
:root {
    --primary-color: #your-color;
    --radius-md: 10px;
    --space-4: 20px;
}
```

## 常见问题

### Q: 界面无法正常显示
A: 请确保：
1. 网络连接正常，能加载Font Awesome和Google字体
2. 使用现代浏览器访问
3. 检查控制台是否有错误信息

### Q: 搜索功能不工作
A: 请检查：
1. 搜索关键词是否正确
2. 是否选择了正确的分类
3. 浏览器是否支持ES6特性

### Q: 命令复制失败
A: 如果复制功能失效：
1. 可以手动选择和复制命令
2. 检查浏览器剪贴板权限
3. 尝试使用不同的浏览器

### Q: 界面显示异常
A: 可以尝试：
1. 刷新页面 (Ctrl+F5)
2. 清除浏览器缓存
3. 禁用浏览器扩展
